Порождающие паттерны:

=============	
	Абстрактная фабрика - паттерн, порождающий объекты.

	Назначение - Предоставляет интерфейс для создания семейств взаимосвязанных или взаимозависимых объектов,
	не специфицируя их конкретных классов.

	Известен также под именем - Kit (инструментарий).
=============
    Фабричный метод
=============
	Одиночка - создает объект в единственном экземпляре, для любого класса клиента
=============
	Прототип — это порождающий паттерн проектирования, который позволяет копировать объекты,
	не вдаваясь в подробности их реализации.
=============
	Строитель - строит различные объекты из набора одинаковых параметров
=============
